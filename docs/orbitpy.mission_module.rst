``orbit.mission`` --- Mission
================================================


Each of the JSON objects in the specifications file has a corresponding OrbitPy object which in turn can be initialized via 
a JSON string containing the appropriate key/value pairs. Hence to know the valid key/value pairs while defining a ``MissionSpecs.json`` file, navigate to
the documentation of the corresponding OrbitPy object API reference. For example the valid key/value pairs of the ``grid`` JSON
object may be found in :ref:`grid_module`. (Description of the instrument objects may be found in the documentation of the 
InstruPy package.)
