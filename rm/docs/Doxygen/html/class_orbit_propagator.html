Detailed Description

This class simulates a Satellite's orbit and calculates access history for ground stations and points of interest.

This class reads from <architecture>.json file and simulates one of the Satellites specified in that file.
As the Satellite's orbit is propagated, access history is calculated for both the Ground Stations and Points of Interest.
This access data can be retrieved for further analysis.

This function will take the parameters for an IntervalEventReport and re-propagate the interval with a smaller interpolation step.

Parameters
MissionEpoch- when the event started
tPropC- the propagation timestep of the correct step
tInpC- the interpolation timestep of the correct step
currEvent- the current event that is being propagated over
eventNum- a counter for which event is currently being processed
totalEvents- the total number of events found
numSuccess- the number of correction steps that found access
numFail- the number of correction steps that didn't find access
Returns
- a vector of Interval Event Reports (should be of length 1)
